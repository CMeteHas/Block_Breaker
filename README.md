# Block_Breaker
```markdown

Block Builder, Python'da Pygame kütüphanesi kullanılarak geliştirilmiş basit bir blok yapım oyunudur. Oyunda, kullanıcılar fare ve klavye kontrollerini kullanarak bloklar ekleyebilir, kaldırabilir ve karakterlerini hareket ettirebilirler.

## Kurulum

1. Öncelikle Python'un yüklü olduğundan emin olun. Python'u indirmek için [Python web sitesini ziyaret edebilirsiniz](https://www.python.org/).

2. Projeyi klonlayın:

```bash
git clone https://github.com/kullanici_adi/proje_adı.git
```

3. Pygame kütüphanesini yükleyin:

```bash
pip install pygame
```

4. Kodu çalıştırın:

```bash
python block_builder.py
```

## Kontroller

- **Fare Sol Tuş (LMB):** Blok eklemek için kullanılır. Her tıklama, bir blok ekler. Eğer tıklanan hücre boşsa, kahverengi bir blok ekler. Kahverengi bir blok varsa, yeşil bir bloğa dönüşür. Yeşil bir blok varsa, hücreyi boşaltır.
- **Fare Sağ Tuş (RMB):** Blok kaldırmak için kullanılır. Tıklanan hücrede bir blok varsa, onu kaldırır.

- **Klavye Ok Tuşları:** Oyuncu karakterini hareket ettirmek için kullanılır.

## Oyun Ekranı

Oyun ekranı, 800x600 piksel boyutunda oluşturulur ve "Block Builder" başlığıyla bir pencerede görüntülenir. Oyun ekranında, oyuncu karakteri ve bloklarla oluşturulan bir alan görüntülenir.
